文字描边效果实现方法

方法1: -webkit-text-stroke (最简单)
实吾云上草
.text {
  color: #ff6b35;
  -webkit-text-stroke: 3px white;
  text-stroke: 3px white;
}
方法2: 多重 text-shadow 模拟描边
原度假乡村
.text {
  color: #ff6b35;
  text-shadow: 
    -2px -2px 0 white, 2px -2px 0 white,
    -2px 2px 0 white, 2px 2px 0 white,
    -2px 0 0 white, 2px 0 0 white,
    0 -2px 0 white, 0 2px 0 white;
}
方法3: 更细腻的多重阴影
精细描边
.text {
  color: #ff6b35;
  text-shadow: 
    -1px -1px 0 white, 1px -1px 0 white,
    -1px 1px 0 white, 1px 1px 0 white,
    -2px -2px 0 white, 2px -2px 0 white,
    -2px 2px 0 white, 2px 2px 0 white,
    -3px -3px 0 white, 3px -3px 0 white,
    -3px 3px 0 white, 3px 3px 0 white;
}
方法4: 伪元素描边
伪元素描边
.text {
  position: relative;
  color: #ff6b35;
}
.text::before {
  content: attr(data-text);
  position: absolute;
  top: 0; left: 0;
  -webkit-text-stroke: 4px white;
  z-index: -1;
  color: transparent;
}
特效1: 发光描边
发光效果
.text {
  color: #ff6b35;
  -webkit-text-stroke: 2px white;
  text-shadow: 
    0 0 10px rgba(255, 255, 255, 0.8),
    0 0 20px rgba(255, 255, 255, 0.6),
    0 0 30px rgba(255, 255, 255, 0.4);
}
特效2: 3D立体描边
立体效果
.text {
  color: #ff6b35;
  text-shadow: 
    1px 1px 0 white,
    2px 2px 0 #ddd,
    3px 3px 0 #ccc,
    4px 4px 0 #bbb,
    5px 5px 10px rgba(0, 0, 0, 0.3);
}
